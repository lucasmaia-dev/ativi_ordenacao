Foram realizados testes com os algoritmos Bubble Sort e Selection Sort, utilizando vetores de tamanhos 100, 1000 e 10000, inicializados em ordem decrescente. A comparação considerou o tempo de execução, o número de comparações e o número de movimentações.

Os resultados mostram que o Bubble Sort apresentou maior número de movimentações, pois realiza várias trocas sempre que encontra elementos fora de ordem. Isso faz com que seu tempo de execução aumente, principalmente para vetores maiores.

Já o Selection Sort apresentou melhor desempenho nas movimentações, pois realiza apenas uma troca por iteração, mesmo mantendo um número semelhante de comparações. Com isso, seu tempo de execução se manteve mais eficiente nos testes.

Dessa forma, conclui-se que o Selection Sort apresentou melhor desempenho geral, sendo mais adequado para vetores de maior tamanho quando comparado ao Bubble Sort.
